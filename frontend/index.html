О проекте

Платформа для анализа IT-рынка и построения карьеры

Граф сходства профессий

Joybs – платформа для студентов и начинающих IT-специалистов. Мы помогаем анализировать рынок, находить востребованные навыки и строить карьерный путь на основе данных.

Метрики: медианная зарплата (RUR, 10k–600k), распределение по группам, частота навыков, влияние навыков на зарплату, коэффициент Жаккара для сходства профессий.

«Найди свой путь в IT через данные»

  • Бэкенд: Python + FastAPI, SQLite
  • Фронтенд: HTML/CSS/JS, Chart.js, D3.js
  • Аналитика: статистические расчёты, матрица навыков, граф сходства
