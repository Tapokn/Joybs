IT Vacancy Analytics

О проекте

Веб-приложение для анализа рынка IT-вакансий на основе данных с hh.ru. Данные содержат ~10 000 вакансий.

Стек: Python FastAPI, SQLite, Chart.js, D3.js, vis-network.

Метрики: медианная зарплата (RUR, 10k–600k), распределение по группам профессий, востребованные навыки, влияние навыков на зарплату.

Граф профессий построен на основе коэффициента Жаккара по навыкам.
